Shorts Factory プライバシーポリシー

yumlia studio（以下「運営者」）は、自社が運営するYouTubeチャンネル向けの動画自動投稿システム「Shorts Factory」（以下「本システム」）における情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

1. 本システムの概要

本システムは、運営者自身が所有・管理するYouTubeチャンネルに対して、動画コンテンツを自動生成・自動アップロードするための内部ツールです。一般公開のウェブサービスやアプリではなく、第三者ユーザーへの提供は行っていません。

2. 収集する情報

本システムでは、以下の情報を取り扱います。

  • YouTube OAuth認証情報：運営者自身が管理するYouTubeチャンネルへの動画アップロード・管理のため、YouTube Data API v3 のOAuth 2.0認証トークンをローカル環境に保存します。
  • チャンネル運用データ：投稿動画のタイトル・説明文・統計情報（再生数・チャンネル登録者数等）を、コンテンツ企画・分析の目的で取得します。

第三者ユーザーの個人情報を収集することはありません。

3. 情報の利用目的

  • 運営者が管理するYouTubeチャンネルへの動画の自動アップロード・予約投稿
  • チャンネルのパフォーマンス分析によるコンテンツ改善

4. 第三者への提供

運営者は、法令に基づく場合を除き、取得した情報を第三者に提供しません。

5. 利用する第三者サービス

  • YouTube Data API Services（Google LLC）
    Google プライバシーポリシー

本システムによるGoogleユーザーデータの利用は、Google API Services User Data Policy（限定使用の要件を含む）に準拠します。

6. データの保管・削除

認証トークンおよび運用データは、運営者が管理するローカル環境にのみ保存されます。データの削除を希望する場合は、以下のお問い合わせ先までご連絡ください。

7. プライバシーポリシーの変更

運営者は必要に応じて本ポリシーを変更できるものとします。変更後のポリシーは本ページに掲載した時点から効力を生じます。
